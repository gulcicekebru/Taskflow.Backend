# Taskflow Backend

This project is a backend API for a simple task management system.

## Purpose
I am building this project to:
- Improve my ASP.NET Web API skills
- Practice clean architecture and layered structure
- Learn authentication, authorization and real-world backend flows

## Current Features
- User authentication (JWT)
- Basic project structure

## Planned Features
- Role-based authorization
- Task CRUD operations
- Task status management
- Validation & error handling

## Status
This project is under development and used as a learning and portfolio project.

> This project is being developed step by step as part of my backend developer journey.
